# MoirAI Sports Engine

Plataforma de inteligência de dados e análise estatística esportiva de alta precisão desenvolvida em TypeScript. Inspirada nas Moiras da mitologia grega, as tecelãs do destino.

## Stack

**Next.js 14** + **React 18** + **TypeScript 5.4** + **decimal.js**

## Scripts

| Comando           | Descrição                        |
| ----------------- | -------------------------------- |
| `npm run dev`     | Inicia servidor de desenvolvimento |
| `npm run build`   | Build de produção                |
| `npm run start`   | Inicia servidor production       |
| `npm run lint`    | Linting                          |
| `npm run typecheck` | TypeScript type checking       |
